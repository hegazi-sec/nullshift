Welcome to NullShift

This wizard walks you through the one-time initial setup. You will configure:

• An admin account (username + password)
• An LLM provider (Claude SDK or API keys later)
• An optional SIEM connector

All settings are written to data/config.db — no .env editing required.

Admin Account

Create the initial administrator login for the NullShift web UI.

LLM Provider

Choose how NullShift connects to an AI model. You can add or change providers at any time from Admin → Settings.

SIEM Connector

Connect to your Security Information and Event Management system. You can skip this and configure connectors later in .env or Admin Settings.

✓

Setup Complete!

NullShift is configured and ready. Start the server with:

uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

If the server is already running, it will continue to work — no restart required for config.db changes.

Open http://localhost:8000 and log in with the admin account you just created.

To add LLM API keys or more SIEM connectors, go to Admin → Settings after login.
