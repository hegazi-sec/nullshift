# Code of Conduct

NullShift is built on **respect, curiosity, and collaboration**. This document describes the standards we expect from everyone interacting with the project — contributors, users, and maintainers alike.

## Our Standards

We want this to be a place where anyone, regardless of background or experience level, feels welcome to participate.

**Examples of behavior that build a healthy community:**

- Being welcoming to newcomers and patient with first-time contributors
- Critiquing ideas, code, and design — never people
- Giving and accepting constructive feedback gracefully
- Owning mistakes, learning from them, and helping others do the same
- Prioritizing what is best for the overall community
- Using inclusive language

**Examples of behavior we will not tolerate:**

- Personal attacks, insults, or derogatory comments
- Public or private conduct that would reasonably be considered inappropriate in a professional setting
- Publishing others' private information without explicit permission
- Sustained disruption of discussions, issues, or pull requests
- Any conduct that makes other participants feel unwelcome

## Scope

This Code applies to all project spaces — the GitHub repository, issues, pull requests, discussions, and any private channels used for project coordination. It also applies whenever someone is officially representing NullShift in a public space.

## Enforcement

If you experience or witness behavior that violates these standards, please contact the maintainer privately at **ahmedislam23@gmail.com**. All reports are confidential and will be reviewed promptly.

The maintainer reserves the right to:

- Remove comments, commits, code, issues, or other contributions that violate this Code
- Temporarily or permanently restrict any contributor whose behavior is inappropriate, threatening, or otherwise unwelcome
- Take any other action deemed necessary to protect the well-being of the community

Decisions are at the maintainer's discretion and will be made with fairness and proportionality in mind.
